This project consists of a main class (Starter.java) and a .jar file, along with 6 subdirectories(package):



lib: Contains the libraries used by the project.
JsonData: Stores tax rates and product category information, which are initialized within the program.
Setup: Contains the Java class for initializing the program with a JSON file.
ProductType: Includes the Java class for creating product objects and saving them in the shopping cart.
CartManagement: Contains the Java class for processing user input, managing objects, and calculating tax rates.
PrintScreen: Includes the Java class for printing out the results.



----------------------------------------------------------------------------------------------------------------------

How to run:

1. open cmd 
2. cd to this file path
3. run with command java -jar interview_test.jar
